# Assessments

This directory contains completed assessments and their results for tracking progress over time.

## Assessment Types

- **Baseline Tests**: Initial knowledge level assessments
- **Progress Checks**: Regular understanding verification
- **Milestone Assessments**: Major checkpoint evaluations
- **Final Evaluations**: Comprehensive subject completion tests

## File Naming Convention

- `{subject}-{test-type}-{date}.md`
- `{subject}-results-{date}.md`
- `{subject}-analysis-{date}.md`

## Assessment Results Format

Each assessment result includes:
- Raw scores and percentages
- Strength/weakness analysis
- Learning plan adjustments
- Next steps recommendations

*Assessment results will be recorded as tests are completed.*
